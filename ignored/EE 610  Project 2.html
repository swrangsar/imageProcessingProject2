Project 2
by Arjun Arunachalam - Thursday, 11 October 2012, 11:11 PM
 
sim_8ch_data.mat

Please find attached data that needs to be used for project 2.

1. The main goal of  the project is to implement the 'A' matrix in the journal paper handed out during project 1 and demonstrate image restoration for differen radon projection values.

2. You must simulate and test the performance of your algorithm for 8, 16, 32 and 64 projections respectively and demonstrate results comparable with that reported in the paper.

3. The anatomy_orig image is the first image for you to demonstrate image restoration on.

4. The other images are weighted images whose weighting functions have also been provided to you. Process them in the manner described in the paper.

5. Finally, try and use the L1 norm constraint with the 2D DCT or wavelets functions in Matlab and see if it helps with any further improvements. This part is optional.

The project is due on Nov 5th.
